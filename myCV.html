Intro

I fell in love with Web Design (HTML and CSS in particular) years ago - in 2007 - if being precise - when it all was much more simple and none of modern tools and libraries existed. But I was fascinated by the logics and simplicity of HyperText MarkUp Language, I liked to create pages "from scratch" and understand what's "inside" the code. At that time I was creating so-called E-books - projects looking like Web Sites by interface and functionality, but instead of being uploaded online their ISO image files were directly burnt to CD-RW discs.

When I moved to US, I realized that I need to learn a new profession (my degree in Linguistics as an English-Russian translator obviously wouldn't be enough to survive here). Being naturally good at languages as well as Math due to my analytical mind I felt like I could learn programming languages - which I did and got a degree in Computer Programming. Now when I have tried a lot of languages and tools, I found that my Main Passion lies in WEB DESIGN, and HTML with CSS are still my favorite ones. :)
